== Description ==

Custom Image Sizes by 99 Robots is a quick and simple way for you to add your own image sizes to your WordPress site.

Also please check out our other [plugins](https://99robots.com/products/?utm_source=wprepo&utm_medium=custom-image-sizes&utm_campaign=desc) :)

== Installation ==

1. Upload `99robots-custom-image-sizes` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to plugins page to see instructions for shortcode and php template tags
